## Technologies

- **Frontend**:
  - React.js
  - TailwindCSS
  - Vite.js
- **Backend**:
  - Node.js
  - Express.js
  - MongoDB
- **Other Tools**:
  - Google Meet API
  - Jest
  - TypeScript
